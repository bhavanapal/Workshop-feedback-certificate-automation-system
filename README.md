# Workshop Feedback Certificate-Automation-System

## Description
A Workshop feedback collection system built with React.js,TypeScript and Context API. Users can submit workshop feedback, and admin can collect student information, verify contact details via OTP, and automatically generate and send personalized certificates to participants.

## Features
- User-friendly responsive UI
- Real-time feedback submission using Appwrite
- State management with Context API
- Admin dashboard to view and manage feedback
- Role-based access control (Admin/User)

## Tech Stack
- **Frontend:** React.js, TypeScript, Context API
- **Backend/Database:** Appwrite
- **State Management:** Context API
- **Tools:** Git, GitHub, VS Code
